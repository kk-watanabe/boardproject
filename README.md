## 学生時代・訓練でのITスキル経験
大学ではFortranを用いてモンテカルロ法による核反応のシミュレーションソフトの開発に取り組みました。物理・数学の知識をもとに計算モデルを構築し、数値計算の精度と処理効率の改善を意識して実装を行いました。特に、計算負荷の高い処理についてアルゴリズムを見直すことで、実行時間の短縮を図りました。

<br>

## 今後の展望
現在は就労移行支援事業所にて、Djangoを用いた図書館アプリや業務アプリの開発に取り組んでいます。基本のCRUD機能の実装に加え、ユーザー操作を意識した画面設計や、エラー発生時の処理、ヴァリデーションなど、実務を想定した開発を意識して取り組んでいきます。今後はさらにAPIを導入し、図書館の蔵書登録の自動効率化まで実装しようと考えています。
